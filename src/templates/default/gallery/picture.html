{* Galerie: Bild zeigen ==================== ACHTUNG: Der komplette HTML-Code einer Website (so wie hier) ist nur dann nötig, wenn die Galerie-Bilder in einem Popup-Fenster gezeigt werden. RESIZE = Javascript-Code um die Fenstergröße anzupassen (wenn das Bild im Popup gezeigt wird) Seitenzahlen ------------ PICTURE_COUNT = Anzahl der Seiten PICTURE_SELECTED = Ausgewählte Seitenzahl PICTURE = Liste der Seitenzahlen PICTURE.NUMBER = Seitenzahl PICTURE.LINK = Link PICTURE_PREVIOUS = Link "vorherige Seite" PICTURE_PREVIOUS_PREVIEW = Pfad zum Thumbnail des vorherigen Bilds PICTURE_NEXT = Link "nächste Seite" PICTURE_NEXT_PREVIEW = Pfad zum Thumbnail des nächsten Bilds PICTURE_FIRST = Link "erste Seite" PICTURE_LAST = Link "letzte Seite" Nur mit Untergalerien --------------------- GALLERY = Liste der Untergalerien GALLERY.ID = Untergalerie-ID GALLERY.TITLE = Untergalerie-Titel GALLERY.LINK = Link zur Untergalerie GALLERY.PREVIEW = Vorschau-Bild GALLERY.PREVIEW_FULLSIZE = Große Version des Vorschau-Bilds GALLERY.DESCRIPTION = Untergalerie-Beschreibung GALLERY.COUNT = Anzahl der enthaltenen Bilder GALLERY.TIME = Zeit (Timestamp) GALLERY.RESTRICTED = Altersabfrage aktiviert?
